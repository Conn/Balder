Las Vegas Style SlotMachine

My contribution to the Norwegian challenge in the European Silverlight Challenge

Built using Silverlight 2

If the game doesn't show up or the installation link for Silverlight 2 doesn't show up. Please download the runtime for Both Windows and Mac and then refresh the page.

This application is a bit graphic-intense and can take a minute or so to download. Depending on your connection speed.


Usage

Give Silverlight focus by clicking your mouse into the Silverlight portion of the screen. When the slotmachine has been downloaded and the download bar has faded out, you press space to start the slotmachine.

The purpose of the game is to get the best combination of symbols.
You get 15 rounds to play and have 2 symbol swaps for each round.
Swapping is done by using the keys 1,2 and 3 to toggle holding of a symbol.
